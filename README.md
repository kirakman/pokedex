## How to Run

Necessary [Node.js](https://nodejs.org/) and NPM installed.

Clone this project, enter the folder and install the dependencies with:

```
npm install
```

And run the project:

```
ng serve
```

Access at [localhost:4200](http://localhost:4200/)
